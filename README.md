## Running the app

- Clone the repository
  ```bash
  $ git clone https://github.com/rakshit-bhalla/gofiber-rest-api-boilerplate.git
  ``` 
- Enter the folder
  ```bash
  $ cd gofiber-rest-api-boilerplate
  ``` 
- Add the configurations and run it using the following command
  ```bash
  $ go run ./src/main.go
  ```
